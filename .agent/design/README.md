# Design Folder

This folder contains the design-system source-of-truth files for Itqan.

Place the design pack files here, including:
- `00_PROJECT_CONTEXT.md`
- `01_DESIGN_FOUNDATION.md`
- `02_DESIGN_RULEBOOK.md`
- `03_V1_SCOPE_AND_EXCLUSIONS.md`
- `04_PRODUCT_COMPONENTS.md`
- `05_SESSION_MODES_SPEC.md`
- `06_SCREEN_ARCHITECTURE.md`
- `07_RTL_IMPLEMENTATION_RULES.md`
- `08_FLUTTER_DESIGN_TRANSLATION.md`
- `09_DESIGN_GAP_REVIEW.md`

These files should be read together with the architecture/context files in `.agent/context/`.
