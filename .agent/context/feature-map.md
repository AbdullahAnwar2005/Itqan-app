# Feature Map

## today
Purpose:
- operational home of the app

Contains:
- today summary
- due tasks
- recovery banner if needed
- resume context
- primary CTA

## plan
Purpose:
- configure memorization and review structure

Contains:
- targets
- weekly plan
- pace adjustment
- active ranges/surahs

## session
Purpose:
- execute memorization, review, and self-test

Contains:
- session mode UI
- Quran content
- audio support
- progress capture
- confidence capture

## progress
Purpose:
- show meaningful progress

Contains:
- summary metrics
- consistency
- review completion
- weak areas
- self-test trends

## library
Purpose:
- supportive browsing/reference

Contains:
- surah browsing
- memorized coverage
- review state markers

## settings
Purpose:
- preferences and app settings

Contains:
- language
- appearance if supported
- audio preferences
- notifications
- about

## shared cross-feature systems
- design system
- routing
- persistence
- localization
- audio
