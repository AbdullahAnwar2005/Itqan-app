# Skill — RTL Arabic UI

## Purpose
Implement Arabic-first interfaces correctly.

## Rules
- Use directional APIs for spacing and alignment.
- Audit icon direction intentionally.
- Right-align Arabic interface text by default where appropriate.
- Keep Arabic typography readable with sufficient line height.
- Use Quran font only for Quran content.

## Avoid
- copying LTR web layout assumptions
- fixed left/right placements in product widgets
- generic body styles for Quran text
