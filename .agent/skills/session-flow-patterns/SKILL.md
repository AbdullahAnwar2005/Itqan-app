# Skill — Session Flow Patterns

## Purpose
Implement Itqan session modes correctly.

## Session modes
- memorize
- review
- selfTest

## Shared rules
- Quran content is the visual center
- assistance controls are secondary
- progress is visible but quiet
- state transitions are explicit
- save/exit behavior should be intentional

## Memorize mode
- chunked flow
- repetition support
- audio support
- confidence capture
- stable / needs-more-work outcomes

## Review mode
- faster flow
- due/overdue awareness
- low-friction issue marking
- continuity-first

## Self-test mode
- reduced assistance
- reveal/hint controls
- result capture
- no flashy quiz styling
